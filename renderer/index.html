Midscene 浏览器助手

先在本机 Chrome 登录系统，再用 Bridge 附加到当前标签；调试完成后用「任务包」分享给同事（仅文本，不含 Key）

任务与业务上下文

「业务上下文」用于多次调试时补充术语、菜单路径、校验规则；与「主任务」会一并发给模型。定稿后复制「任务包」即可微信转发。

浏览器连接（登录态）

推荐 Bridge：在桌面 Chrome 安装 Midscene 扩展，先手动打开网站并登录，再在本工具点运行。扩展会自动连到本机端口（默认 3766）。

Windows 示例：先关闭所有 Chrome，再运行
"C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222
然后在 chrome://version 或启动日志里找到 DevTools 的 ws 地址粘贴到此处。

模型配置

API Key 仅存本机，不会进入任务包。同事需各自配置自己的 Key。
