# 谷歌浏览器自动化助手（基于Midsence 桌面自动化）

## 技能功能
这是一个基于桌面视觉自动化的工具，用于**操控谷歌浏览器**完成人类无法用 API 实现的桌面操作，例如：
- 比如打开我的财务系统，自动填写报销单（测试阶段用例为浏览百度新闻）
- 浏览器页面点击、输入、滚动、截图识别
- 任何需要鼠标/键盘操作的网页流程自动化

## 支持平台
- macOS（当前开发环境）
- Windows（可通过 Electron 打包分发）

## 核心能力
1. 视觉定位：通过屏幕图像识别网页元素
2. 鼠标操作：单击、双击、右键、拖拽
3. 键盘操作：输入文本、快捷键
4. 流程编排：用自然语言生成自动化脚本
5. 跨平台：macOS / Windows 双系统支持

## 调用规则
- 用户描述任务（自然语言）→ AI 生成自动化脚本
- 使用用户提供的视觉模型 + 文本推理模型 openai API Key
- 所有操作仅作用于谷歌浏览器
- 支持保存、复用、编辑自动化任务

## 禁止行为
- 不访问恶意网站
- 不窃取用户数据
- 不执行未经用户确认的高危操作

## Skills

- planning-with-files: 复杂任务、调研任务、或者预计会有 5 次以上工具调用时使用。
- context7-cli: 需要最新库文档、API 参考或 Context7 示例时使用。
- agent-browser: 需要操作 Google Chrome、打开网页、填写表单、点击按钮、截图、抓取数据、自动化财务系统报销单等浏览器任务时使用。支持自然语言直接驱动（如“帮我登录财务系统并填写报销单”）。
- browser: Midscene 浏览器自动化
- browser-use: 需要持久浏览器会话、持续监控页面状态、多次交互或复杂流程时使用。
- electron: 后续打包成 Electron 桌面应用、自动化 Electron 打包后的工具自身，或跨平台桌面自动化时使用（macOS/Windows 均支持 CDP）。
